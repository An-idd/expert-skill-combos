# 目的
把已写好的学术文稿打磨得更清楚、更规范:语言、逻辑、引用格式。只润色已有内容,不替你写、不造数据。

# 何时使用
- 已有初稿(论文/综述/开题),要改语言、理顺逻辑、统一引用。
- 专家在工作流程第 4 步调用。

# 输入
- 待润色文稿
- 目标引用格式(GB/T 7714 / APA / MLA / 期刊要求)

# 步骤
1. 语言:去口语化和冗余,术语一致,长句拆短,被动/主动得当(常见问题见 [references/citation-styles.md](references/citation-styles.md))。
2. 逻辑:每段一个中心、段间过渡、论点—证据—出处闭环。
3. 论断查源:每个关键论断有没有引用支撑?无源论断标出来让作者补。
4. 引用规范:正文引注与参考文献列表按目标格式统一,查漏补缺。
5. 学术诚信自检:有没有无意抄袭(大段照搬未标注)、引用是否与原意相符。
6. AIGC 风险自查:综述/总结段是否像 AI 平铺?引导用自己的话重写、加原创判断(2026 高校/期刊查 AI 率,见 [references/aigc-awareness.md](references/aigc-awareness.md))——帮你写得像真人真思考,不是教你骗检测。

# 输出
- 润色后文稿(改动可标注)
- 修改说明(语言/逻辑/引用各改了什么)
- 待作者补的清单(缺出处的论断、格式存疑的引用)

# 边界
- 不代写实质内容、不编造数据或引用。
- 不做同行评审级的学术判断;涉及方法/统计正确性,建议找导师或专业人士。
