# 综述组织方式 + 研究缺口类型

> 用于 review-outline。

## 综述的几种组织方式(选一个主线,别流水账)
- 主题式:按核心议题分节(最常用)。
- 时间式:按研究演进的时间线。
- 方法式:按研究方法/流派分。
- 理论式:按理论框架分。
- 对立式:围绕一个争议的正反两方组织。

## 研究缺口的常见类型
- 人群/情境缺口:某类对象/场景没人研究。
- 方法缺口:现有研究方法有局限(样本小、缺纵向、缺对照)。
- 结论矛盾:文献结论打架,需要新研究调和。
- 变量缺口:某个调节/中介变量没被考察。
- 时效缺口:环境变了,旧结论需重新验证。

## 系统综述规范(PRISMA 简版)
做系统性文献综述时,按规范流程并留痕:
1. 检索:明确检索式和数据库。
2. 筛选:按纳入/排除标准去重、筛标题摘要、读全文。
3. 记录:各阶段命中/排除数量(可画流程图),保证可复现。
4. 分析:提取数据、评估质量(见 paper-digest)、归纳。
普通课程综述不必严格 PRISMA,但"检索-筛选-纳入"的透明度值得借鉴。

## 一句话自检
读完你的综述,读者能不能清楚说出"这个领域已知什么、还缺什么、你要补哪块"?
