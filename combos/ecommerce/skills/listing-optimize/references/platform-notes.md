# 电商平台差异速查

> 用于 listing-optimize。各平台算法、人群、规则不同,别一套打天下。

## 平台特性
- 淘宝/天猫:搜索 + 推荐并重;直通车/万相台;人群标签精细;重视 DSR 评分。
- 拼多多:极致性价比 + 推荐流;多多搜索/场景推广;低价起量;重视销量与好评。
- 京东:自营 + POP;京速推;3C/家电心智强;重视物流与正品信任。
- 抖店:内容 + 直播驱动的兴趣电商;搜索占比低于货架;靠短视频/直播引流。

## 标题/关键词习惯
- 天猫:品牌词 + 品类 + 属性 + 场景长尾。
- 拼多多:核心词 + 强性价比词,简单直给。
- 抖店:口语化、场景化,配合内容标签。

## 通用红线
- 每个平台都有极限词/虚假宣传/违规促销规则,违规扣分甚至封店。
- 别跨平台照搬:同一商品在拼多多打低价、在天猫打品质,策略不同。
