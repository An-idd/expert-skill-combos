# 目的
把"找达人带货/种草"变成有筛选标准、有谈判要点、有 ROI 跟踪的合作方案。

# 何时使用
- 要通过达人种草/带货拉新,或搭建达人素材供给。
- 专家在"内容/流量"环节调用(常在大促蓄水期配合 campaign-plan)。

# 输入
- 商品/品牌、预算、目标人群、合作目标(种草/直播带货/素材)

# 步骤
1. 定类型:头部(打声量)/腰部(性价比)/尾部 KOC(铺量真实感),按预算配比。
2. 筛达人:看粉丝画像匹配度、真实互动率(防刷)、内容调性、带货历史(清单见 [references/influencer-checklist.md](references/influencer-checklist.md))。
3. 谈合作:报价形式(坑位费/纯佣/框架)、排期、内容要求、复用授权。
4. 要素材:约定可二次分发的素材(星图),沉淀成品牌内容资产。
5. 跟 ROI:按 GMV/曝光/涨粉核算,留数据决定要不要复投。

# 输出
- 达人分层配比 + 候选清单(带匹配理由)
- 合作方案(形式/排期/内容要求/授权)
- ROI 跟踪表模板

# 边界
- 核实达人数据真实性,防刷粉刷量;不做虚假宣传/违规带货。
- 合作不代表转化保证,按数据迭代。
