# 目的
把一场大促从"临时促销"变成有节奏、有机制、有预算、能复盘的完整战役。

# 何时使用
- 备战 618/双11/双12/年货节等大促。
- 专家在工作流程第 2 步调用。

# 输入
- 大促名称与日期、目标(GMV/拉新/清库存)
- 商品与库存、可用预算、历史大促数据

# 步骤
1. 排节奏:提前 45-60 天启动,分 蓄水期 → 预售期 → 爆发期 → 返场期(玩法见 [references/promo-mechanics.md](references/promo-mechanics.md))。
2. 设机制:预售/定金膨胀、跨店满减、店铺券、赠品、阶梯价。
3. 分预算:流量(直通车/信息流)、折扣让利、达人/直播 三块怎么切。
4. 备货与客服:按预估销量备库存(防超卖掉分),大促期客服/物流扩容。
5. 留后手:大促客户进留存漏斗(会员/私域),别做一次性买卖。

# 输出
- 活动节奏表(各阶段时间/动作/目标)
- 机制设计(优惠玩法 + 力度)
- 预算分配表
- 备货与客服预案
- 复盘指标清单(大促后对照)

# 边界
- 守毛利红线,不为 GMV 亏本冲量。
- 不设计虚假折扣(先涨后降)、不违规。
