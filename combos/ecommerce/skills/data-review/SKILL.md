# 目的
把一堆经营数据拆成"哪儿好、哪儿漏、下一步做什么",而不是只看个 GMV 数字。

# 何时使用
- 周期性经营复盘,或大促/活动结束后。
- 专家在工作流程第 4 步调用。

# 输入
- 经营数据(GMV、订单、客单价、转化率、流量来源占比、ROI、退货率、复购率)
- (可选)对比基准(上期/去年同期/目标)

# 步骤
1. 看大盘:GMV = 流量 × 转化率 × 客单价,先定位哪个因子拖后腿。
2. 拆流量结构:免费搜索/付费/推荐/直播/外部 各占多少、健不健康(过度依赖付费 = 危险)。
3. 算账:付费 ROI、单件毛利、退货率,别只看 GMV 不看赚没赚。
4. 找漏点:哪个环节转化掉得多(曝光 → 点击 → 加购 → 下单 → 支付)。
5. 给动作:每个问题对应一条可执行的下一步(优化 listing/调价/改投放/改活动)。

# 输出
- 大盘诊断(GMV 三因子哪个是短板)
- 流量结构评估(健康度 + 风险)
- 盈利评估(ROI/毛利/退货)
- 转化漏斗漏点定位
- 下一步动作清单(问题 → 动作)

# 边界
- 用真实口径,不美化数据;基准因品类差异大,不套用别处的固定值。
- 只做经营分析,不替代财务核算。
