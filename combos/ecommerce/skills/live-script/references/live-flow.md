# 直播带货话术与节奏库

> 用于 live-script。

## 单品话术五步
痛点唤起 → 卖点演示 → 场景带入 → 价格锚点(原价 vs 直播价)→ 限时逼单。

## 排品逻辑
- 引流款:低价/福利,开场聚人气、拉停留。
- 利润款:主推,靠话术和演示转化。
- 爆款:高认知度,承接流量高峰。
- 冲量款:临近结尾,限量憋单冲 GMV。

## 逼单手法(合规前提下)
限时(倒计时)、限量(仅 X 单)、专属价(仅直播间)、赠品(前 X 名)、涨价预告。

## 互动拉权重
福袋/抽奖留人、扣屏(扣"想要")测意向、问答解决顾虑、宠粉福利提复访。

## 红线
不虚假原价比价、不喊极限词、承诺的价格赠品必须兑现。
