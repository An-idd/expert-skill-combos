# 目的
把一场直播从"随便播"变成有节奏、有话术、有逼单点的转化脚本。

# 何时使用
- 要做一场直播带货,或直播转化低要优化脚本。
- 专家在工作流程第 3 步调用。

# 输入
- 直播商品清单(卖点/价格/利润/库存)
- 直播时长、主播风格、目标(GMV/涨粉/清库存)

# 步骤
1. 排品:引流款(低价聚人)→ 利润款 → 爆款 → 冲量款,穿插节奏(话术结构见 [references/live-flow.md](references/live-flow.md))。
2. 搭话术:每个品按"痛点—卖点—演示—价格锚点—限时逼单"讲。
3. 设逼单点:限时/限量/专属价/赠品,踩整点和人气高峰放大。
4. 设计互动:福袋、抽奖、扣屏、问答,拉停留和权重。
5. 兜控节奏:开场留人、中场高潮、结尾憋大招,别平铺。

# 输出
- 商品排序表(每品时长 + 定位)
- 每品话术卡(痛点/卖点/演示/价格锚点/逼单话术)
- 逼单与互动节点时间轴
- 开场/结尾话术

# 边界
- 不喊违规极限词、不虚假比价、不夸大功效。
- 价格/赠品承诺要能兑现,别翻车。
