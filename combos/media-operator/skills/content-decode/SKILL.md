# 目的
把"别人为什么爆"拆成你能直接套用的模板。输出模板,不输出成稿。

# 何时使用
- 选定选题方向后、动笔前。
- 想搞清某类笔记的爆款公式。
- 专家在工作流程第 2 步调用。

# 输入
- 3-10 条同赛道对标爆款(链接或截图,尽量含标题/首图/正文/点赞收藏数)

# 步骤
1. 逐条拆五个维度:
   - 标题钩子:用了什么公式(数字/悬念/身份代入/痛点/反差)。
   - 首图/封面:排版、文字、色调套路。
   - 正文结构:开头怎么抓、中间怎么铺、结尾 CTA 怎么收。
   - 标签:话题标签数量与组合(大词+中词+长尾)。
   - 发布时间:星期与时段。
2. 横向找共性:哪些套路在多条里反复出现(共性 = 可复用信号)。
3. 剔除偶然因素(靠达人带/投流的,不代表内容本身能复制)。
4. 提炼成一套"可套用模板"。

# 输出
1. 拆解表(每条爆款 × 五维度)。
2. 一套可复用模板:
   - 标题公式(填空版,如「__岁也能__的__」)
   - 正文骨架(分段提示)
   - 标签组合建议
   - 建议发布时段
3. 一句话:这个模板最适合什么选题、什么时候别用。

# 边界
- 区分"内容套路"和"外部助推"(达人/投流),只提炼可自主复制的部分。
- 样本 <3 条时提醒结论不稳,建议补样本。
