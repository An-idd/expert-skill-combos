# 小红书爆款结构库

> 用于 content-decode 快速识别一篇爆文属于哪种结构、为什么有效。

## 常见爆款结构类型
1. 清单体:「N 个__」——信息密度高,易收藏
2. 教程体:「保姆级 / 手把手」——实用,收藏率高
3. 踩坑体:「别买 / 避雷 / 我的学费」——恐惧驱动,点开率高
4. 测评体:「亲测 / 连用 N 天」——真实感,信任度高
5. 故事体:「从__到__」——代入感,完读率高
6. 对比体:「A vs B」——决策辅助,转化好
7. 反差体:「别再__了」——冲突制造好奇

## 拆解五维度(逐条爆文过一遍)
- 标题钩子:用了哪种公式(见 script-writer/references/hooks.md)
- 首图 / 封面:排版、文字、色调套路
- 正文结构:开头怎么抓、中间怎么铺、结尾怎么收
- 标签:大词 + 中词 + 长尾组合与数量
- 发布时间:星期与时段

## 判断可复制性
剔除靠达人 / 投流的外部助推,只提炼内容本身可自主复制的套路。样本 <3 条结论不稳。
