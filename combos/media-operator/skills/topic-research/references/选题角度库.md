# 选题角度库 + 热点来源

> 用于 topic-research 给同一主题快速拓多个切口,并知道去哪找热点。

## 选题角度类型(同一话题换切口)
- 痛点型:大家最头疼的那个问题(脱妆 / 塌头 / 踩雷)
- 对比型:A vs B、平价 vs 大牌、新手 vs 老手
- 清单型:N 个 / 一篇讲清 / 合集
- 避坑型:别买 / 我交的学费 / 踩过的雷
- 教程型:手把手 / 保姆级 / 步骤
- 测评型:亲测 / 连用 N 天 / 翻车实录
- 反常识型:其实 / 别再 / 你以为的是错的
- 场景型:通勤 / 约会 / 换季 / 出差 场景化
- 情绪型:嘴替 / 爽文 / 治愈 / 破防

## 热点来源清单
- 平台内:搜索下拉词、话题榜、同赛道近 7 天高赞新笔记
- 季节节点:换季、节日、开学季、大促、节气
- 站外:抖音 / 微博热搜同主题、行业新品发布
- 评论区:自己和对标账号评论区的高频提问 = 现成选题

## 优先级判断(各 1-5 分)
账号相关度 × 当前热度 ÷ 竞争饱和度。相关度 <3 直接砍。
