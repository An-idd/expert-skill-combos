# 小红书合规/限流参考清单(2026)

> 经验归纳,非官方,规则会变——以平台最新规则为准。用于发布前自查。

## 1. 极限词 / 绝对化用语(机审高危,易直接限流)
最、最好、第一、No.1、绝对、100%、纯天然、根治、治愈、永不复发/反弹、
全网最低、行业第一、国家级、顶级、独一无二……
→ 改法:换成相对表达("我用下来""个人体感""据说"),去掉绝对承诺。

## 2. 2026 新禁夸张词(平台阶段性收紧)
绝了、封神、天花板、必入、王炸、yyds、神器……
→ 改法:用具体感受替代情绪词("回购了三次""比之前那支润")。

## 3. 引流 / 导流词(外链违规,封号高发区)
私信我、加微信/vx/v 我、加群、主页链接、公众号、扣 1、滴滴我、看简介……
→ 改法:去掉硬导流;需要留资走平台内合规方式。

## 4. 模糊夸大副词滥用
超、很、非常、巨、爆、狂 —— 堆砌显营销、机审不喜。
→ 改法:少用副词,用事实和细节撑说服力。

## 5. AI 造假 / 真实性红线(2026 平台核心)
虚构使用体验、无实物摆拍宣称、搬运洗稿、明显 AI 腔。
→ 改法:真人真实体验、真图真感受;宁可朴素,不要假。

## 6. 行业敏感(按赛道另查)
医疗/功效(祛痘根治、抗癌)、金融(稳赚、保本)、教育(保过、包分配)——
这些赛道另有专项审核,涉及时提醒用户走合规或找专业人士。
