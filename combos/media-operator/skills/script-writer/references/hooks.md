# 小红书标题钩子公式库

> 用于 script-writer 生成标题、content-decode 对标分析。示例仅示范结构,按赛道替换。

## 万能结构
[人群] + [痛点] + [解决方案] + [好奇钩子]
例:「油皮姐妹|夏天脱妆脸油?这套哑光流程扛 8 小时」

## 常用钩子类型
1. 痛点直击:「__总是__?可能是你__」
   例:洗完头两天就塌?可能是你护发素涂错位置
2. 数字清单:「__的 N 个__」
   例:平价护肤别踩雷,我交过学费的 5 个坑
3. 身份代入:「__岁/__党 也能__」
   例:学生党 100 块搞定一整套通勤穿搭
4. 悬念留白:「__到底__?我试了__才知道」
   例:贵妇面霜到底值不值?我连用 30 天才敢说
5. 反差冲突:「别再__了,__才是__」
   例:别再狂敷面膜了,过度护肤才是烂脸元凶
6. 结果前置:「__天/__次,__的变化」
   例:早睡 21 天,素颜被问用了什么护肤品

## 立钩子的原则
- 0.3 秒抓情绪:痛点 / 好奇 / 反差 三选一,别平铺直叙。
- 关键词前置:目标人群能搜到的词放前面(利于搜索流量)。
- ≤20 字,首图 + 标题共同决定点开率。
- 合规优先:钩子再强也不碰极限词/夸张违禁词(见 compliance-check)。
