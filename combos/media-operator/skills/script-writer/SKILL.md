---

# 目的
把选题 + 模板变成可直接发布的成稿。这是执行的最后一步。

# 何时使用
- 已有选题(来自 topic-research)和模板(来自 content-decode)。
- 专家在工作流程第 3 步调用。

# 输入
- 选题 + 角度
- 爆款模板(标题公式/正文骨架/标签建议)
- 账号语气(真人感/专业/搞笑…)

# 步骤
1. 标题:按模板公式出 3-5 版,覆盖不同钩子(痛点版/数字版/悬念版),每版 ≤20 字。
2. 正文:按骨架写,开头 2 行内立钩子,口语、真人感,分段短、留呼吸感,结尾一个明确 CTA。
3. 标签:大词+中词+长尾组合,数量按模板建议。
4. 封面文案:一句主标 + 可选副标,≤12 字,一眼看懂利益点。
5. 自检:标题有钩子?正文是生活方式框架而非硬卖?CTA 清晰?违规词?

# 输出
- 标题:3-5 版(标注各自钩子类型)
- 正文:成稿
- 话题标签:一组
- 封面文案:主标 + 副标
- 一句自检结论(过没过上面 5 条)

# 边界
- 不编造亲测效果、成分功效、绝对化承诺(违规且掉信任)。
- 医疗/金融/功效类,加一句"建议用户核实/合规审核"。
- 只产文案,不改选题方向(要往回退就该找专家重判)。
