# 目的
把"找客户"从无差别海投,变成信号驱动、个性化的精准开发。

# 何时使用
- 要开发新客户、搭建触达序列。
- 专家在工作流程第 1 步调用。

# 输入
- 产品/解决方案、目标市场
- (可选)现有成交客户特征、可用渠道

# 步骤
1. 定 ICP:理想客户画像(行业/规模/角色/痛点),分层(高/中/低契合,见 [references/outreach-playbook.md](references/outreach-playbook.md))。
2. 找信号:什么信号说明该客户此刻可能有需求(招聘、融资、上新、政策)。
3. 设计序列:多渠道(邮件/电话/社交)组合,几次触达、间隔多久。
4. 写破冰:个性化开场,基于该客户的具体信号,不群发模板。

# 输出
- ICP 定义 + 客户分层标准
- 触达序列(渠道 × 次数 × 间隔 × 每次说什么)
- 个性化破冰话术(几版)

# 边界
- 个性化基于真实信息,不编客户情况;遵守骚扰/隐私规则。
- 质量优先于数量,不做无差别轰炸。
