# 职场邮件场景话术

> 用于 email-writer。

## 通用原则
- 主题直给内容 + 期望动作。
- 开头一句说清目的和要对方做什么。
- 正文分点、重要信息前置;一封信一件主事。
- 结尾明确 next step(谁/何时/做什么)。

## 分场景
- 请求/协作:说清背景 → 具体请求 → 为什么找他 → 截止。
- 汇报:结论先行 → 关键进展 → 需要的支持。
- 跟进催办:友好提醒 → 重申事项和截止 → 降低对方回复成本。
- 邀约:目的 → 时间选项 → 议程 → 回复方式。
- 拒绝:先谢/共情 → 明确拒绝并给理由 → 给替代方案或留余地。

## 语气
对上:尊重简洁、给选择;对平级:协作互惠;对外:专业得体。

## 忌
主题含糊("关于那件事")、正文冗长绕圈、没有明确的行动项、情绪化措辞。
