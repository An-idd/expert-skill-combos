# 目的
把一节课的内容变成一份可上课用的教案:目标明确、重难点清楚、环节可操作。

# 何时使用
- 要备一节新课,或优化现有教案。
- 专家在工作流程第 1 步调用。

# 输入
- 学段/年级/学科、课题(章节)
- (可选)所用教材版本、课时、学情

# 步骤
1. 定位:这节课的核心知识点、在单元中的位置。
2. 学情:学生认知起点、易错点(没提供就基于学段常识,并标注为假设)。
3. 三维目标:知识与技能、过程与方法、情感态度,目标要可观测(用"能说出/能运用"等行为动词,见 [references/teaching-design.md](references/teaching-design.md))。
4. 重难点:结合课标和学情确定。
5. 教学环节:导入 → 新授 → 练习 → 小结 → 作业(五环节),每环节设计梯度问题和学生活动。

# 输出
- 教学目标(三维,可测)
- 教学重难点
- 教学环节设计(每步:教师活动/学生活动/设计意图/预设问题)
- 板书/作业要点

# 边界
- 对齐课标但以现行教材为准;学情假设要标出。
- 出教案供参考,课堂实施和最终定稿归教师。
