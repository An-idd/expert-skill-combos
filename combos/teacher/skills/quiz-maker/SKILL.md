# 目的
按教学目标和知识点出一套对应的题,带答案解析,难度可控。

# 何时使用
- 要出随堂练习、作业、单元测。
- 专家在工作流程第 2 步调用。

# 输入
- 学段/年级/学科、考查的知识点范围
- (可选)题量、题型、难度分布、总分

# 步骤
1. 对知识点:每道题明确考哪个知识点和目标。
2. 配题型与难度:客观题(选择/填空)测识记理解,主观题测运用分析(认知层次见 [references/item-design.md](references/item-design.md))。
3. 控难度分布:易/中/难按需配比(如 7:2:1)。
4. 出答案解析:每题给答案 + 解析(为什么、易错点)。
5. (可选)双向细目表:知识点 × 认知层次 × 分值 一览。

# 输出
- 题目(按题型分组)
- 答案 + 解析(含易错点)
- 难度分布说明
- (可选)双向细目表

# 边界
- 题目对齐知识点,不超纲;拿不准的标注。
- 开放题给评分要点而非唯一答案,不硬造有争议的"标准答案"。
