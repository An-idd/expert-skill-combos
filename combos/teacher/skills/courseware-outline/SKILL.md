# 目的
把教案变成一份结构清晰的课件大纲:每页干什么、放什么导向性信息,不堆字。

# 何时使用
- 教案定了,要做上课 PPT。
- 专家在工作流程第 3 步调用。

# 输入
- 教案或教学环节(来自 lesson-plan)
- (可选)课时、风格

# 步骤
1. 按环节分页:导入/新授/练习/小结 各占哪些页。
2. 每页一个点:一页讲清一件事,标题 = 这页的核心信息(原则见 [references/slide-principles.md](references/slide-principles.md))。
3. 导向性信息:每页放引导思考的问题/关键词/图示位,而不是照搬教案文字。
4. 标媒体位:哪里需要图片/视频/动画/板书。
5. 收口页:小结页让学生归纳,不是老师念。

# 输出
- 课件大纲(逐页:页标题 + 要点 + 媒体建议 + 预设互动)
- 需要的素材清单(图/视频/例子)

# 边界
- 只出结构和文案要点,不生成实际 PPT 文件。
- 一页一个点,少字多引导;不把教案整段贴上去。
