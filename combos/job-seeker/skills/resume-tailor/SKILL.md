name: resume-tailor
description: 按岗位拆解结果定制简历——最相关证据前置、责任句改成果句、对齐 ATS 关键词,并诚实标出缺口。绝不编造经历。
---

# 目的
把通用简历改成对准这个岗位的定制简历。只对真实经历动刀,不造假。

# 何时使用
- 已有目标岗位的 jd-decode 结果 + 求职者真实简历。
- 专家在工作流程第 2 步调用。

# 输入
- 求职者真实简历(或经历素材)
- jd-decode 的拆解结果(必须项/关键词/评估点)
- (可选)目标语气、篇幅限制

# 步骤
1. 重排:把最能对上必须项的经历/项目/技能前置,让招聘方前 6 秒就看出匹配。
2. 改写 bullet:责任句 → 成果句,套"动作 + 范围 + 量化结果 + 业务影响"。缺数字就留占位问句(问用户),不编。
3. 对齐关键词:在真实支撑下融入 JD 术语,同一关键词在不同语境出现 2-3 次(别同处堆砌——2026 ATS 语义匹配,堆砌反被扣分);软技能嵌进成果示例。规则见 [references/ats-rules.md](references/ats-rules.md)。
4. 标缺口:必须项里没证据支撑的,诚实标出,并给真实补法(相邻经历/项目/课程/证书/作品集/求职信角度)。
5. 每处大改附一句理由:改了什么、对应哪条要求、为什么更强。

# 输出
- 定制后简历(或改动版块)
- 改动说明清单(改动 → 对应要求 → 为什么更强)
- 缺口清单 + 真实补法
- 待求职者确认的占位问句(缺的量化数据)

# 边界
- 绝不编造工作/学历/证书/数据/项目/成果。
- 关键词不堆砌、不靠一次接触吹成熟练。
- 不用花哨表格/图形/多栏,保证 ATS 可解析。
