# 目的
把初稿改到能送审:格式规范、用语准确、逻辑通顺、无错字。

# 何时使用
- 已有初稿,要校对定稿。
- 专家在工作流程第 3 步调用。

# 输入
- 待校对初稿
- (可选)单位公文格式要求

# 步骤
1. 结构逻辑:层次是否分明、段旨是否清楚、有无重复或断裂(检查清单见 [references/proofread-checklist.md](references/proofread-checklist.md))。
2. 用语规范:公文语体、称谓、规范表述,去口语和浮夸。
3. 格式:标题、发文字号、层次标号、附件、落款等按党政机关公文格式核对。
4. 事实核查:数据、政策、人名地名单位名核对,存疑标出。
5. 文字:错别字、标点、病句。

# 输出
- 校对后文稿(改动可标注)
- 问题清单(结构/用语/格式/事实/文字 分类)
- 待作者核实项(缺出处或存疑的内容)

# 边界
- 不改变原意和事实;拿不准的标出让作者定,不擅自改数据。
- 格式以本单位/最新规范为准。
