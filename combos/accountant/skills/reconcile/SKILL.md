# 目的
把账面和实际(银行、对方、明细)对平,每个差异都查出原因。对账是侦探活,不是走过场。

# 何时使用
- 月末结账前的例行对账,或发现某科目余额异常。
- 专家在工作流程第 2 步调用。

# 输入
- 待对账科目的账面数据 + 外部凭据(银行对账单/往来对账单/盘点表)

# 步骤
1. 定范围:这次对哪些科目(清单见 [references/reconcile-checklist.md](references/reconcile-checklist.md))。
2. 逐项比:账面余额 vs 外部数,列出差异明细。
3. 查差异:未达账项(在途)、记账错误、遗漏、时间性差异分别归因。
4. 出调整建议:该补的分录、该核对的凭证。
5. 留痕:差异原因和处理写清,便于复核和审计。

# 输出
- 对账表(科目/账面/外部/差异)
- 差异清单 + 归因
- 待处理事项(补分录/核凭证/催对方确认)

# 边界
- 不为对平硬凑数;查不出的差异如实标"待查",不掩盖。
- 金额无论大小,原因不明都要查(重要性决定紧急度,不决定是否查)。
