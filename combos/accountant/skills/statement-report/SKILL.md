# 目的
结账后出一套准确、勾稽得上的财务报表,并做基本的变动分析,让报表能支持决策。

# 何时使用
- 月末结账完成后编报表。
- 专家在工作流程第 4 步调用。

# 输入
- 结账后的试算平衡表/科目余额(来自 month-close)
- (可选)上期报表、预算、目标

# 步骤
1. 编资产负债表:资产 = 负债 + 所有者权益,按科目归集填列(结构与勾稽见 [references/statement-notes.md](references/statement-notes.md))。
2. 编利润表:收入 - 成本费用税金 = 利润,逐级列示。
3. 勾稽校验:表内(资产=负债+权益)、表间(利润表本年利润 → 资产负债表未分配利润)对得上。
4. 变动分析:环比/同比/预算差异,大幅波动给解释。
5. 出结论:一句话说清这期经营状况(盈亏、现金、异常)。

# 输出
- 资产负债表
- 利润表
- 勾稽校验结果(平/不平 + 差异定位)
- 环比/预算差异分析 + 波动解释

# 边界
- 数据来自已结账账簿,不美化、不倒挤。
- 报表编制不等于审计;对外披露/税务申报以专业人士和监管口径为准。
- 现金流量表等更复杂报表如需要,建议找专业会计处理。
