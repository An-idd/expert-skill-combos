# 财务报表结构 + 勾稽关系

> 用于 statement-report。以企业适用会计准则的报表格式为准。

## 资产负债表(时点)
- 恒等式:资产 = 负债 + 所有者权益。
- 资产:流动资产(货币资金、应收账款、存货…)+ 非流动资产(固定资产净值、无形资产…)。
- 负债:流动负债(应付账款、应交税费、应付职工薪酬…)+ 非流动负债。
- 权益:实收资本、盈余公积、未分配利润。

## 利润表(期间)
营业收入 - 营业成本 - 税金及附加 - 期间费用(销售/管理/财务)
+ 其他收益等 = 营业利润;± 营业外收支 = 利润总额;- 所得税费用 = 净利润。

## 关键勾稽
- 表内:资产合计 = 负债合计 + 权益合计。
- 表间:利润表"净利润"结转 → 资产负债表"未分配利润"变动。
- 现金:货币资金变动应能被经营/投资/筹资活动解释。

## 不平怎么查
倒查:结转分录是否完整、有无科目漏填、方向错、余额取错期。
