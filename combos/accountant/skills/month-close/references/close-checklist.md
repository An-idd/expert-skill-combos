# 月末结账清单

> 用于 month-close。按顺序过一遍,别漏项。

## 结账前
- [ ] 本月所有业务已入账(发票/回单/费用报销)
- [ ] 各科目已对账,差异已处理(见 reconcile)
- [ ] 银行、现金、往来余额核实

## 计提项(常见)
- [ ] 工资、社保、公积金
- [ ] 固定资产折旧、无形资产摊销、待摊费用
- [ ] 税金及附加(城建税、教育费附加等)
- [ ] 水电、利息等应计未付
- [ ] 增值税结转(销项 - 进项,月末转应交)

## 结转
- [ ] 成本结转(库存商品 → 主营业务成本)
- [ ] 期间损益结转本年利润

## 结账后
- [ ] 试算平衡(借贷相等)
- [ ] 关键科目环比,异常查因
- [ ] 凭证装订、留痕

## 提醒
计提/税务处理以企业适用准则和最新税法为准;不确定的找财务负责人或税务师确认。
