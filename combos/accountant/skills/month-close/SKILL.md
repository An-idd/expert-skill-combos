# 目的
把一个月的账按流程结掉:该计提的计提、该结转的结转、试算平衡,准时且准确。

# 何时使用
- 每月月末结账。
- 专家在工作流程第 3 步调用。

# 输入
- 本月已入账凭证 + 对账结果(来自 reconcile)
- (可选)结账日历、上月数据

# 步骤(按结账清单,见 [references/close-checklist.md](references/close-checklist.md))
1. 补录:确认本月业务都入账了、对账差异都处理了。
2. 计提:工资、社保公积金、折旧摊销、税金、水电等应计未付项。
3. 结转:成本结转、损益结转(收入/费用结转本年利润)。
4. 试算平衡:借贷合计相等、各科目余额合理。
5. 复核:关键科目余额与预期/上月对比,异常波动查因。

# 输出
- 计提分录清单
- 结转分录清单
- 试算平衡表
- 异常波动说明(如有)
- 结账完成确认(哪些做了、哪些待跟进)

# 边界
- 准确不可妥协、快是加分;不为赶进度跳过对账或复核。
- 手工调整分录必须有摘要、依据、审批,"调整分录"不算摘要。
- 不擅自调整以前期间(如需更正,披露影响)。
