# 常见业务记账分录 + 科目对照

> 用于 bookkeeping。示例为一般思路,具体科目/税率以企业适用会计准则和最新税法为准。

## 常见业务分录(思路)
- 采购原材料(取得增值税专票):
  借:原材料 / 应交税费—应交增值税(进项税额)　贷:银行存款 / 应付账款
- 销售商品(开专票):
  借:银行存款 / 应收账款　贷:主营业务收入 / 应交税费—应交增值税(销项税额)
- 结转销售成本:
  借:主营业务成本　贷:库存商品
- 支付费用(房租/水电/办公):
  借:管理费用 / 销售费用—明细　贷:银行存款
- 计提工资:
  借:管理费用 / 生产成本—工资　贷:应付职工薪酬
- 发放工资(代扣个税/社保):
  借:应付职工薪酬　贷:银行存款 / 应交税费—应交个人所得税 / 其他应付款—社保
- 往来收付款:
  借/贷:应收账款 / 应付账款 与 银行存款 对冲

## 常用一级科目
- 资产:库存现金、银行存款、应收账款、其他应收款、原材料、库存商品、固定资产、累计折旧
- 负债:应付账款、其他应付款、应交税费、应付职工薪酬
- 权益:实收资本、盈余公积、本年利润、利润分配
- 损益:主营业务收入、主营业务成本、税金及附加、管理费用、销售费用、财务费用

## 提醒
税率、科目设置随准则/政策变化,以最新为准;小企业会计准则与企业会计准则在部分科目上有差异。
