# 目的
把一笔经济业务正确记成会计分录:科目对、借贷平、有原始凭证支撑。面向中小企业/个体日常做账。

# 何时使用
- 日常发生业务(采购、销售、费用、收付款、工资等)要入账。
- 专家在工作流程第 1 步调用。

# 输入
- 经济业务描述 + 原始凭证(发票/银行回单/合同等)
- (可选)企业适用准则(小企业会计准则/企业会计准则)、行业

# 步骤
1. 判性质:这是什么业务(采购/销售/费用/资产/往来/税金…)。
2. 选科目:借贷双方各挂哪个会计科目(常见业务分录见 [references/common-entries.md](references/common-entries.md))。
3. 编分录:借贷金额相等,附摘要说清"什么事"。
4. 挂凭证:每笔分录关联原始凭证编号,无票的说明来源。
5. 涉税提示:涉及增值税/附加税的,标出销项/进项和适用税率(以最新税法为准)。

# 输出
- 记账凭证(借方科目/贷方科目/金额/摘要)
- 关联的原始凭证说明
- 涉税事项提示(如有)

# 边界
- 不做假账、不虚构业务或票据。
- 科目/税率以企业适用准则和最新税法为准;拿不准的标出让用户与财务/税务确认。
- 复杂或特殊业务(股权/并购/跨境)建议找持证会计师。
